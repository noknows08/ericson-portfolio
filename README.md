# Ericson Sombrea — Personal Portfolio

Multi-page personal portfolio website for **Ericson Sombrea**, IT Engineer / System Engineer based in Davao City, Philippines.

## Pages

| Page | File |
|------|------|
| Home / About | `index.html` |
| Professional Experience | `experience.html` |
| Skills & Certifications | `skills.html` |
| Contact | `contact.html` |

## Stack

- Plain HTML5 / CSS3 / vanilla JavaScript — no build step required
- Google Fonts (Sora + Inter)
- Deployed on [Vercel](https://vercel.com) as a static site (`vercel.json` enables clean URLs)

## Local preview

Just open `index.html` in a browser, or serve the folder:

```powershell
npx serve .
```

## Deploy

```powershell
vercel deploy --prod
```
